## [509. 斐波那契数](https://leetcode-cn.com/problems/fibonacci-number/)

[Fibonacci number](https://en.wikipedia.org/wiki/Fibonacci_number)

### Iterative实现思路

```
F(0) = 0,   F(1) = 1
F(N) = F(N - 1) + F(N - 2), for N > 1.
```

计算`F(N)`仅仅依赖于前两个元素。
